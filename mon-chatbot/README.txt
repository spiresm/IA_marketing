# Mon Chatbot GPT

## Étapes pour lancer le projet

1. Ouvrir un terminal dans ce dossier
2. Installer les dépendances :
   npm install

3. Lancer le serveur :
   npm start

4. Ouvrir un navigateur à :
   http://localhost:3000/chatbot.html

## ⚠️ Important

Remplacer la clé API dans server.js :
apiKey: 'VOTRE_CLÉ_API_OPENAI_ICI'
